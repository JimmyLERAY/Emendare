### Import

`import { Column } from '../../../components';`

### Basic Usage

```jsx
import { Column, Columns } from '../../../components'
;<Columns>
  <Column className="has-background-light">First column</Column>
  <Column className="has-background-light">Second column</Column>
  <Column className="has-background-light">Third column</Column>
</Columns>
```
